# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

## [v1.0.0] - 2026-01-08
- Versão final inicial do SimpleCalc.
- Funcionalidades: soma, subtração, multiplicação, divisão, potência, módulo, raiz quadrada, porcentagem (produto/100).
- Observações: usa delays de 2s para suspense; entrada por `int()`; possíveis erros com divisão por zero e raiz de número negativo.
