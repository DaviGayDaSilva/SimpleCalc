# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

## [v1.0.0] - 2026-01-08
- Versão final inicial do SimpleCalc.
- Funcionalidades: soma, subtração, multiplicação, divisão, potência, módulo, raiz quadrada, porcentagem (produto/100).
- Observações: usa delays de 2s para suspense; entrada por `int()`; possíveis erros com divisão por zero e raiz de número negativo.

## [v1.1] - 2026-26-02
- Adicionada proteção contra divisão por zero (não quebra mais o programa).
- Adicionada divisão inteira (//) mostrando quantas vezes um número entra no outro.
- Adicionado cálculo de módulo (%).
- Adicionado fatorial (n!) com proteção para números negativos.
- Adicionado número aleatório entre os dois números.
- Adicionado valor absoluto (abs()) de cada número.
- Adicionada média aritmética dos dois números.
- Adicionado teste de par ou ímpar para cada número.
- Adicionado cálculo do MDC (máximo divisor comum).
- Adicionado cálculo do MMC (mínimo múltiplo comum).
- Adicionada raiz cúbica de cada número.
- Adicionada tabuada de 1 a 10 de cada número.
